# Rest Integração

### Preço - Precificação  
Projeto de inclusão de um novo preço para um produto e filial  
Run PrecificacaoApplication.java http://localhost:9081/swagger-ui.html  

## Arquitetura

* RestAPI
* Java 8
* Maven
* Spring boot / Mvc / Event
* Tomcat embedded
* CouchDB
* Hibernate
* Swagger
* Fluent Http
